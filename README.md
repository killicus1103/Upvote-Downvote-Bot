# Upvote-Downvote-Bot

a simple discord bot for generating a upvote/downvote reaction on any message.

## set up

to set up, you need to create an auth.json and config.json file.

example auth.json
 ```
 {
“token”: "generatedToken”
}
 ```
 
 example config.json
 ```
 {
	"prefix": "!",
	"token": "generatedToken"
}
 ```
 
 to generate a token, contact me.
